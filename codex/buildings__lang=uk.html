Відмовити Прийняти
Політика конфіденційності
Крамничка
Збудуйте свою власну крамничку у цій області. Вона доволі часто оновлюює доступний асортимент предметів для покупки.
✓ Придатний для будівництва
Бестіарій
Збудуйте свій власний бестіарій у цій області. Він дозволить вам найняти собі компаньйона.
✓ Придатний для будівництва
Кузня
Збудуйте свою власну кузню у цій області. Вона використовується для удосконалення зброї та іншого спорядження.
✓ Придатний для будівництва
Ярмарка
Збудувати ярмарку. На ярмарці торгується величезна кількість товарів, та побудова оної чудово замінює велику кількість побудованих крамничок.
✓ Придатний для будівництва
Цитадель
Збудуйте цитадель у вашому рідному місті. Цитаделі можуть зберігати як золото, так предмети і компаньйонів.
✓ Придатний для будівництва
Фортеця
Перетворіть свою цитадель у фортецю, щоб збільшити ліміти зберігання золота та компаньйонів та отримати доступ до алхіміка та підземелля.
✓ Придатний для будівництва
Замок
Перетворіть свою фортецю у замок, щоб збільшити ліміти зберігання золота та компаньйонів та отримати доступ до демонгарту та колізею.
✓ Придатний для будівництва
Мерія
Збудуйте мерію у вашому рідному місті. Вона потрібна для зацікавлення містян.
✓ Придатний для будівництва
Оплот
Збудуйте Оплот та отримуйте підношення від ваших вдячних містян. Чим більше населення міста та чим краще настрій містян, тим більша вірогідність отримати в дар від них щось цікавеньке.
✓ Придатний для будівництва
Хатинка
Будуйте хатинки, щоб збільшити населення вашого рідного міста.
✓ Придатний для будівництва
Травник
Найняти травника у цій області. Він може варити усілякі потужні еліксири.
✓ Придатний для будівництва
Храм оракула
Побудувавши храм у рідному місті, ви можете отримати послуги досвідчених елізійських оракулів. Вдячні оракули, час від часу, можуть принести вам мнемогліф, а також у разі необхідності, можуть перетворити ваші зайві пам'ятки на новий мнемогліф.
✓ Придатний для будівництва
Оцінювач
Найняти оцінювача предметів для цієї області. Він зможе ідентифікувати знайдені предмети.
✓ Придатний для будівництва
Рибацька хатинка
Побудуйте рибацьку хатинку. Місцевий рибалка із задоволенням обміняє вашу рибу на корали, а також може запропонувати свої особливі товари.
✓ Придатний для будівництва
Гільдія Спелеологів
Побудуйте зал Гільдії Спелеологів. Завзяті дослідники печер зможуть обміняти тут здобуті цінні мінерали.
✓ Придатний для будівництва
Гільдія Монументалістів
Побудуйте зал Гільдії Монументалістів, щоб отримувати нагороди за участь у ігрових подіях.
✓ Придатний для будівництва
Майстерня Титанів
Побудуйте Майстерню Титанів, що знадобиться як для зведення Олімпійських веж, так і для розблокування особливих класів та зброї.
✓ Придатний для будівництва
Брама шляху
Побудуйте Браму шляху для подорожей між рідними містами членів вашої ватаги. Крім того Брама шляху дозволяє миттєво відвідати власне рідне місто у разі потреби.
✓ Придатний для будівництва
Гільдія Мандрівників
Побудуйте зал Гільдії Мандрівників, щоб прийняти участь у церемоніальних випробуваннях Гільдії.
✓ Придатний для будівництва
Зал гільдії Завойовників
Побудуйте зал гільдії Завойовників для отримання здобичі з захоплених поселень.
✓ Придатний для будівництва
Рубаки Фінесса
Збудуйте зал гільдії Рубак Фінесса, щоб змагатися у двобоях у реальному часі з іншими членами гільдії.
✓ Придатний для будівництва
Гільдія Авантюристів
Збудуйте зал Гільдії Авантюристів, щоб створювати пригодницькі завдання для інших шукачів пригод.
✓ Придатний для будівництва
Коло Мучеників
Побудуйте Коло Мучеників у своєму місті, якщо жадаєте більш складних пригод на своєму шляху.
✓ Придатний для будівництва
Вівтар Вознесіння
Збудуйте Вівтар Вознесіння щоб жертвувати богам ваші коштовності в обмін на силу піднесення.
✓ Придатний для будівництва
Застава
Додаток до вашої цитаделі, що дозволяє отримати доступ до компаньйонів та предметів у цитаделі - та якщо ви знаходитесь далеко від неї.
✓ Придатний для будівництва
Ювелір
Побудуйте ювелірну майстерню у цій області. Ювеліри продають прикраси, а також прикрашають зброю та обладунки.
✓ Придатний для будівництва
Цех переробки
Побудуйте цех переробки у цій області. Він знадобиться для перетворення одних матеріалів на інші.
✓ Придатний для будівництва
Трактир
Завітайте до гостинного двору для того, щоб виконати кілька завдань. Також, ви маєте змогу дізнатися про цікавинки неподалік.
Алхімік
Завітайте до цієї будівлі задля майстергартування вашого спорядження. Така дія підіймає рівень предмету з 10 до 11. Також, тут можна оздобити предмети прикрасами, або зачарувати зброю базовими стихіями.
Демонгарт
Завітайте сюди задля демонгартування вашого спорядження. Ця дія підіймає рівень предмета з 11 до 12.
Базар
Базар, доволі рідкісна будівля, має унікальний асортимент предметів, доступних для придбання.
Колізей
Для справді серйозного випробування - прийміть участь у серії поєдинків із іншими героями у Колізеї.
